# fis3 template

## 目录说明

- src/components    存放pug组件
- src/css           存放stylus样式
- src/font          存放字体
- src/images        存放图片
- src/js            存放js文件
- src/lib           存放第三方依赖
- src/mixin         存放stylus mixin
- src/view          存放页面视图

## 本地开发

`fis3 server start && fis3 release -wL`

## 打包构建

`fis3 release prod -d output`
